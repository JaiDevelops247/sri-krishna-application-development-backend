# React and Spring Boot Course : Backend Lab Solutions

This repository contains the complete lab solutions for the Spring Boot
backend section of the full stack course. Each lab is on a separate branch.
Branches are released after each lab session.

## Released Labs

| Lab | Branch | Concepts |
| --- | ------ | -------- |
| Lab 04 | lab-04-spring-boot-rest-api | Spring Boot, REST, CRUD, Postman |
| Lab 05 | lab-05-spring-security-jwt | Spring Security, JWT, Filter Chain, AuthenticationManager, BCrypt |
| Lab 06 | lab-06-oauth2-implementation | OAuth2, Spring Authorization Server, Resource Server, Grant Types, Scope Enforcement |
## Upcoming Labs

Branches for upcoming labs will be shared after each session concludes.

## Prerequisites

- Java 17 or above
- Maven (bundled with IntelliJ)
- IntelliJ IDEA Community Edition
- Postman desktop application

## How to Run Any Lab

1. Switch to the lab branch
2. Open the project in IntelliJ IDEA
3. Wait for Maven to download dependencies
4. Run StudentApiApplication.java
5. Server starts on the port specified in application.properties
